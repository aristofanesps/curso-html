usando o flexbox pra deixar textos, imagens, videos lado a lado

  • display: flex; deixa lado a lado texto, imagem, video.
  • o flex-direction: row; deve ser usado logo abaixo do display: flex; para assim a configuração do conteudo ocorrer na horizontal
  • justify-content: flex-start; posiciona os elementos pra esqueda
  • justify-content: flex-end; posiciona os elementos pra direita
  • justify-content: center; posiciona os elementos no centro do eixo
  • justify-content: space-between; deixa os elementos espaçados automaticamente
  • justify-content: space-around; deixa os elementos espaçados, a diferença é que deixa ka com margem
  • o gap; vc pode definir o tamanho de espaçamento que deseja por, ex: gap: 16px;
